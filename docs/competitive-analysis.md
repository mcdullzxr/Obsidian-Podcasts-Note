## 一、主流竞品全景

| 产品 | 形态 | 核心输出 | 生态 |
|------|------|---------|------|
| **通义听悟** | Web | 关键词、概要、章节速览、发言总结、问答回顾、带发言人逐字稿 | 阿里独立平台 |
| **Snipd** | iOS/Android | AI 章节、Snip（金句+音频+摘要）、嘉宾信息、提到的书、Chat with Episode | 导出到 Notion/Obsidian/Readwise |
| **Podwise** | Web | 大纲、金句、思维导图、关键词、逐字稿 | 导出 Markdown |
| **飞书妙记** | IM 内置 | 发言人、章节、待办、AI 摘要、评论批注 | 飞书内 |
| **Readwise Reader** | Web/App | 高亮 → 笔记 | 导出 Obsidian/Notion |
| **AudioPen/Granola** | Web/App | 原始录音 + 整理后笔记（双轨） | 独立 |

---

## 二、各家独有亮点

### 通义听悟
- 🏷️ 关键词（可点击跳转）
- 📑 章节速览（时间戳 + 标题 + 摘要）
- 👥 发言总结（每个人说了什么）
- ❓ 问答回顾（Q&A 对卡片）
- 🎵 底部音频播放器常驻

### Snipd
- ✂️ Snip（耳机三击捕获）——音频片段 + 转录 + 摘要三位一体
- 📚 Mentioned Books 自动提取
- 👤 Guest profiles
- 💬 Chat with Episode
- 🎯 Shareable quote cards

### Podwise
- 🗺️ 思维导图
- 💎 Highlights 独立页

### 飞书妙记
- 💬 评论/批注（团队协作）
- ✅ AI 提取待办项

---

## 三、我们的差异化优势（Obsidian 原生能力）

1. **知识点 → 原子笔记 + 双链**：竞品都是孤岛
2. **嘉宾/播客/书籍作为 [[双链实体]]**：自动构建知识网络
3. **Dataview 仪表盘**：本月听过哪些播客、涉及 AI 话题的节目…
4. **关联现有 Vault 笔记**：LLM 可读取相关笔记，给出交叉引用 insight

## 四、路线图

### v0.2 — 追平通义基础体验
- [ ] 音频本地下载 + 时间戳点击跳转（`#t=秒数`）
- [ ] 发言人识别（火山 `enable_speaker_info` / 通义 `diarization_enabled`）
- [ ] 章节速览（每章一句摘要）
- [ ] Q&A 卡片提取

### v0.3 — Obsidian 独占
- [ ] 知识点原子笔记 + 双链
- [ ] 嘉宾/书籍作为 `[[双链实体]]`
- [ ] Dataview 仪表盘模板

### v0.4 — 惊艳功能
- [ ] Chat with Episode（侧边栏对话）
- [ ] Mentioned Books/Tools 自动识别
